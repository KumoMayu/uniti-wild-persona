# UniTI

**UniTI** 是一个面向大学年龄段用户的人格测试小站。

它不尝试扮演正式心理量表，而是用更轻的方式贴近当代大学生的日常状态：学业压力、就业焦虑、人际关系、情绪波动、适应问题、信息流干扰、拖延、临时爆发、稳定推进，以及“明明知道该做什么，但还是会先等等”的生活现场。

这个项目想做的不是给出标准答案，而是给出一个有辨识度、能引发共鸣、又保留一定现实感的结果。

## Features

- 24 题人格测试
- 16 型人格结果页
- 题目与选项随机
- 结果图导出
- Cloudflare Pages 静态部署
- 最小聚合统计：总完成人数与各人格命中次数

## Model Overview

UniTI 不直接套用传统人格测试框架，而是关注大学生生活环境下常见的四种底层倾向：

- **节奏倾向**：更喜欢提前铺垫，还是更容易卡点启动
- **能量模式**：更适合稳定续航，还是更依赖波动爆发
- **状态来源**：更靠自己消化整理，还是更容易被环境和他人带动
- **生活姿态**：更偏收着活、求稳，还是更偏放着活、重体验

这些维度不是为了追求绝对准确，而是为了描述现实生活中更常见的差异：有人稳，有人拖；有人慢热，有人临场起飞；有人自己消化一切，有人很吃氛围；有人能长期控节奏，有人更适合边晃边活。

## Scoring Logic

当前版本采用 **四维累计 + 原型匹配** 的评分方式。

系统会先根据用户答题结果累计四个内部维度分数，再将这组分数标准化，并与 16 型人格的原型画像做距离匹配，输出最接近的结果。

这样做的原因是：现实中的人很少刚好落在某个极端标签上，更多时候会处在中间地带、摇摆地带，或者呈现某几维明显、某几维混合的状态。相比直接按维度正负硬切，原型匹配通常更自然，也更方便后续根据真实样本继续调参和校准。

当前模型还不是经过大样本验证的正式量表。它更像一个有现实基础、可继续修正的测试原型。

## Project Structure

关键文件如下：

- `src/pages/wild-persona.astro`：测试页面入口
- `src/data/wildPersonaQuestions.ts`：24 题题库
- `src/data/wildPersonaTypes.ts`：16 型人格文案与配置
- `src/scripts/wildPersonaApp.ts`：前端答题与结果逻辑
- `src/utils/scoring.ts`：四维累计与人格原型匹配
- `src/utils/share.ts`：结果图生成
- `functions/api/track-complete.js`：记录完成测试
- `functions/api/stats.js`：返回聚合统计

## Cloudflare Pages

项目保持 Cloudflare Pages 静态站路线。

如果需要启用统计功能，在 Cloudflare Pages 中绑定一个 KV namespace，变量名设为 `STATS`。统计接口只记录总完成人数与 16 型人格命中次数，不记录用户答案，不做登录，也不保存用户明细。

## Current Status

当前版本小范围公测中。

仍在继续优化的部分主要包括：

- 评分机制调参
