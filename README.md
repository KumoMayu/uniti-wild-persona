# UniTI

**UniTI / 大学生野生人格图鉴** 是一个面向大学年龄段用户的人格测试小站。

它不假装是正式心理量表，也不提供心理学诊断；它更像一个轻量、可转发、带一点校园生活观察的测试，用 24 道题描述当代大学生常见的节奏、能量、社交状态和生活姿态。

## 当前能力

- 24 题测试，按 4 个维度分组
- 16 型人格结果页
- 竖版分享图导出
- Cloudflare Pages 静态部署
- Pages Functions + KV 的最小后台统计
- 开发模式下的评分调试和随机分布模拟

## 算法说明

当前版本采用 **四维累计 + 原型匹配**，而不是直接按四个维度的正负硬切。

用户答完 24 题后，系统会先累计：

- `rhythm`：节奏倾向
- `energy`：能量模式
- `social`：状态来源
- `posture`：生活姿态

每个维度会被标准化到 `-100 ~ 100`，再与 16 型人格的原型向量做加权曼哈顿距离匹配，输出距离最近的结果。这样能更自然地处理现实里常见的中间态和混合态。


## Cloudflare Pages

项目保持 Cloudflare Pages 静态站路线。统计功能使用 Pages Functions + KV，KV binding 名称为 `STATS`。

当前接口：

- `POST /api/track-complete`：记录总完成人数和人格命中次数
- `GET /api/stats`：返回总完成人数和 16 型人格计数

## 项目结构

- `src/data/wildPersonaQuestions.ts`：24 题题库
- `src/data/wildPersonaTypes.ts`：16 型人格结果文案
- `src/data/wildPersonaPrototypes.ts`：人格原型和维度权重
- `src/utils/scoring.ts`：评分与原型匹配
- `src/utils/share.ts`：分享图生成
- `src/scripts/wildPersonaApp.ts`：前端交互逻辑
- `functions/api/track-complete.js`：完成统计上报
- `functions/api/stats.js`：统计读取
